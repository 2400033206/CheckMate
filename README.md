# CheckMate

## Overview

CheckMate is a web-based accountability system designed to increase daily consistency by combining partner-based accountability and financial stakes. Users commit to a 30-day challenge, choose a stake between £50 and £1000, and invite a trusted partner (friend or family member) to join.

## Features

- **Financial Stakes**: Set monetary stakes to incentivize commitment
- **Partner Accountability**: Share your goals with a trusted partner
- **30-Day Challenges**: Structured, time-bound goals
- **Web-Based**: Easy access from any device
- **User-Friendly Interface**: Intuitive design for seamless experience

## Getting Started

### Prerequisites

- Python 3.8+
- pip (Python package manager)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/YOUR_USERNAME/CheckMate.git
cd CheckMate
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Run the application:
```bash
python app.py
```

## Usage

1. Start the application
2. Create an account or log in
3. Set your 30-day challenge
4. Select your financial stake
5. Invite a partner to join
6. Begin your accountability journey!
